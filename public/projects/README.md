# Project screenshots

Drop a screenshot here named by the project `id` and it appears automatically
on that project's carousel card and in its detail view. Until a file exists, a
branded placeholder is shown (no broken images).

- Format: `.jpg`, 16:9 (e.g. 1600×900), keep under ~300 KB
- Filename = the project id from `lib/content.ts`:

| file                      | project              |
| ------------------------- | -------------------- |
| `dreamtrip.jpg`           | DreamTrip.mn         |
| `khan.jpg`                | Khan Consulting      |
| `tender.jpg`              | Tender Monitor       |
| `suvdandusal.jpg`         | Suvdandusal Automation |
| `legal.jpg`               | Online Legal Platform |
| `smm.jpg`                 | Digital Marketing & SMM |
